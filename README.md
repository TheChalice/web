## 使用方法

1. 克隆该项目到本地
```shell
git clone https://github.com/lanyuechen/angular-seed.git
```

2. 安装依赖（本地需安装npm+bower）
```shell
npm install
bower install
```

3. 测试
```shell
npm test
```

4. 开始
```shell
npm start
```
访问：localhost:8000/app
